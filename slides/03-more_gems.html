Front End

fontawesome

awesome vector icons for our application 


  # Gemfile

  gem 'font-awesome-sass'

example 


  <%= icon 'edit', 'Edit' %>

Gmaps4Rails

integrate GoogleMaps in our app 


  # Gemfile

  gem 'gmaps4rails'

example 


  @users = User.all
  @hash = Gmaps4rails.build_markers(@users) do |user, marker|
    marker.lat user.latitude
    marker.lng user.longitude
  end
              

  markers = handler.addMarkers(<%=raw @hash.to_json %>);

Paperclip

attach files to our models 


  # Gemfile

  gem 'paperclip'

example 


  has_attached_file :avatar
              

  <%= image_tag user.avatar.url %>

Gettext

i18nalize our text strings 


  # Gemfile

  gem 'gettext_i18n_rails'

example 


  
<%= _('New Product') %>

Back End

CanCanCan

authorization made simple 


  # Gemfile

  gem 'cancancan'

example 


  if @producer.can? :edit, @product

Geocoder

addresses to LatLng and viceversa 


  # Gemfile

  gem 'geocoder'

example 


  geocoded_by :address
              

  Producer.near 'Grahamstown', 20

Delayed Jobs

manage our async tasks 


  # Gemfile

  gem 'delayed_job_active_record'

example 


  @user.delay.activate!
